Software

 Desktop Web Browsers 
  •   
    Tor Browser ★
    The Tor Browser is one of the safest browsing experiences by prioritizing anonymity & freedom of information above all else. It's one of the most effective and accessible anonymity tools.

    • Open Source
    • Firefox Base
    •
  •   
    Firefox
    Firefox doesn't have the best defaults, reserving maximum privacy & security through hardening the browser. An attractive reason to use Firefox is it's an alternative to the Chromium monopoly.

    • Open Source
    • Firefox Base
    •
  •   
    LibreWolf ★
    LibreWolf tackles most Firefox hardening out-of-the-box & does a decent job of maintaining security updates. Be aware that using forks of projects come with concerns regarding trusting a new party, in addition to trusting them to maintain security updates.

    • Open Source
    • Firefox Base
    •
  •   
    Brave ★
    Brave attempts to offer a familiar browser experience with privacy & security by default. Brave does many neat things for users out-of-the-box, but a possible annoyance is the cryptocurrency features. (Can be disabled)

    • Open Source
    • Chromium Base
    •
  •   
    Ung-Chromium
    Ungoogled-Chromium aims to strip the remaining bits of Google from the open source Chromium browser. Security updates are a bit slower, and it has some (out-of-the-box) usability sacrifices.

    • Open Source
    • Chromium Base
    •
  •   
    Safari
    Apple's Safari offers users in the Apple ecosystem a moderately safe default browser with basic privacy & security precautions out-of-the-box.

    • Proprietary
    • WebKit Base
